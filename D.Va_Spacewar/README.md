# 传送

条件：
1. 存活(事件玩家) == 真
2. Y方向分量(位置(事件玩家)) <= 100

动作：
1. 传送(事件玩家, 矢量(随机实数(-100, 100), 600, 随机实数(-100, 100)))
2. 等待(0.25, 当“假”时中止)
3. 如条件为“真”则循环
